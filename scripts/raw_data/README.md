# Raw Data

This directory contains raw data files collected from Spotify.

## Contents

- `df_musinque.csv` - Raw Spotify playlist data collected using the data collection scripts

## Description

Raw data files in this folder represent unprocessed Spotify playlist information including:
- Playlist IDs and names
- Curator information
- Follower counts
- URLs
- Track counts
- Other metadata

## Data Flow

1. **Collection**: Data is collected using scripts in `../data_collection/`
